# Project Files

This folder contains all the executable project files.

## Files Included:

1. **raw_sales_data.csv** - Original sales data
2. **processed_data.csv** - Cleaned data ready for analysis
3. **data_preparation.py** - Script to clean raw data
4. **analysis.py** - Script to analyze data
5. **app.py** - Flask web server
6. **index.html** - Dashboard web page (in templates folder)
7. **style.css** - Styling (in static folder)
8. **requirements.txt** - Python dependencies
